## 🎵 BGMクレジット
本作品では以下のフリー音源を使用させていただいております。

- **1_2_3_4.ogg (8bit_Burning-Heart)**
  提供：ノザワ音楽工房 様 (https://yuki-nozawa.com/)

- **Pause.ogg (8bit_Everyday)**
  提供：ノザワ音楽工房 様 

- **Gokuen.ogg (8bit-maou-kessen_Loop)**
  提供：もみじばミュージック 様 (https://music.storyinvention.com/)

  ## 注意点について
  「読み込みをSkip」ボタンは通信環境が整ってない方に向けてBGMの読み込みをスキップし、ゲームを始められるようにするためのものです
  読み込みが完了している場合はどちらを押しても挙動は変わりません
